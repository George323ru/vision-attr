# Vision Attractor

BI-дашборд для анализа жизненных ценностей. Интерактивный граф показывает иерархию «аттракторов» (жизненных доменов) и корреляции между ними. Правая панель отображает предиктивный анализ для выбранных категорий.

---

## Стек

| | |
|---|---|
| Фреймворк | Vue 3 + TypeScript |
| Сборщик | Vite |
| Граф | vis-network 9 |
| Данные | JSON (fetch при старте) |

---

## Структура данных

**11 доменов** — жизненных сфер (Быт, Независимость, Переживания, Работа, Саморазвитие, Семья, Социум, Стабильность, Тело, Убеждения, Увлечения).

Каждый домен содержит трёхуровневую иерархию аттракторов:

| Уровень | Кол-во | Описание |
|---|---|---|
| L1 | 11 | Корневые узлы (по домену) |
| L2 | 57 | Базовые аттракторы, точки анализа |
| L3 | 444 | Конкретные поведенческие паттерны |

**137 корреляций** между L2-аттракторами с возрастной динамикой (усиление / конфликт).

```bash

npm run dev        # http://localhost:5173
```

```bash
npm run build      # сборка → dist/
```

Данные загружаются из `public/data/attractors.json` при старте. Исходник — `.data/attractor_dataset.csv` (2944 строки). Для пересборки JSON:

```bash
python3 scripts/parse_csv.py
```

## Возможности

**Граф**
- Раскрытие/свёртывание узлов кликом (L1 → L2 → L3)
- Мульти-фокус: клик выделяет узел и все его корреляции
- Серые рёбра корреляций показываются фоном (переключатель в шапке)
- Режимы: развернуть все L2 / все L3

**Правая панель**
- Фильтры демографии: возраст (слайдер диапазона), пол, семейное положение, дети
- 3 дропдауна выбора L2-аттракторов для сравнения
- Корреляции между выбранными аттракторами + топ-10 внешних
- Подсветка корреляций на графе (зелёный = усиление, красный = конфликт)
- 31 жизненная ситуация с предиктивным расчётом стратегий по возрасту

**Возрастная динамика**
- Каждая корреляция имеет возрастные диапазоны и силу связи
- При движении слайдера возраста граф пересчитывает толщину и цвет рёбер

## Структура проекта

```
src/
├── App.vue                    # корневой компонент
├── composables/
│   ├── useAppState.ts         # весь UI-стейт (синглтон)
│   ├── useNetwork.ts          # управление vis-network (инстанс)
│   ├── useAttractorStore.ts   # данные аттракторов (синглтон)
│   ├── useVisualSettings.ts   # настройки графа (синглтон)
│   ├── useCorrelations.ts     # расчёт корреляций (stateless)
│   └── usePrediction.ts       # предиктивный расчёт (stateless)
├── components/
│   ├── NetworkCanvas.vue      # canvas-контейнер графа
│   ├── RightPanel.vue         # правая панель с роутингом
│   ├── AttractorDropdowns.vue # 3 дропдауна + панель корреляций
│   ├── DemographicsPanel.vue  # слайдер возраста, фильтры
│   └── panels/                # AttractorPanel, SituationPanel, L3Panel, ...
├── data/
│   ├── correlations.ts        # 137 корреляций
│   ├── situations.ts          # 31 ситуация
│   └── themes.ts              # цвета для vis-network canvas
└── utils/
    ├── nodeStyles.ts          # стили узлов + wrapLabel()
    ├── edgeStyles.ts          # стили рёбер
    └── colors.ts              # HSL-расчёт цветов по домену

public/data/
└── attractors.json            # домены + аттракторы (загружается fetch())
```
