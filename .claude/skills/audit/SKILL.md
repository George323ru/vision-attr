# Codebase Audit

Проведи структурированный аудит кодовой базы и выдай отчёт.

## Шаги

1. Используй Task-агенты для параллельного анализа:
   - **Типы**: запусти `npx vue-tsc --noEmit`, собери все ошибки типизации
   - **Качество кода**: найди unused imports, dead code, дублирование, несогласованные паттерны
   - **Архитектура**: проверь соответствие паттернам из CLAUDE.md (singleton composables, vis-network интеграция, тройная иерархия)
2. Для каждой найденной проблемы укажи: файл, строку, описание, приоритет (critical/medium/low)
3. Сгруппируй по категориям: Types, Code Quality, Architecture, Performance
4. НЕ исправляй ничего автоматически — только отчёт
5. Выведи отчёт в консоль в формате markdown
